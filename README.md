# TimeTravel Agency - Webapp Interactive

Bienvenue dans TimeTravel Agency, une webapp immersive et luxueuse dédiée à une agence de voyage temporel fictive. Conçue comme un projet universitaire mêlant design premium, expérience utilisateur et intelligence artificielle générative, elle permet d’explorer des destinations extraordinaires à travers le temps.

Cette expérience interactive a été développée pour proposer une visite élégante et narrative, avec une interface moderne et des fonctionnalités intelligentes au service de l’immersion.

Membres du projet : Enzo Nyssen, Adriana Pullig, Mickael Smyczyk, Louis-Alexandre Chavalle 

## 🛠️ Stack Technique

- Framework : React (initialisé via Vite)
- Styling : Tailwind CSS
- Animations : Framer Motion
- Icônes : Lucide React
- IA / Modèle de langage : Mistral AI API
- Déploiement : Vercel

## ✨ Fonctionnalités Implémentées

- Landing page interactive au design premium avec mode sombre et accents dorés
- Galerie dynamique des 3 destinations temporelles (Paris 1889, Crétacé, Florence 1504) avec modales d’informations détaillées
- Chatbot IA conversationnel intégré sous forme de widget flottant, avec un agent configuré comme expert en histoire
- Moteur de recommandation interactif sous forme de quiz dynamique

## 🤖 Outils IA Utilisés (Transparence)

- Génération de code : GitHub Copilot (VS Code)
- Moteur du chatbot : Mistral Small (via API)
- Visuels et assets : images et concepts visuels générés avec DeeVid AI lors de la Session 1

## 🚀 Instructions d’Installation

```bash
# 1. Clonez le repository
git clone <url-du-repo>
cd TimeTravelAgency

# 2. Installez les dépendances
npm install

# 3. Configurez la clé API dans un fichier .env
echo "VITE_MISTRAL_API_KEY=votre_cle_api" > .env

# 4. Lancez le serveur de développement
npm run dev
```
